# HSGS Hackathon 2025

## Cách chạy hệ thống chấm điểm (judge):

### 1. Tạo môi trường ảo:
```bash
python3 -m venv venv
```

### 2. Kích hoạt môi trường ảo:

**Trên Linux / macOS:**
```bash
source venv/bin/activate
```

**Trên Windows:**
```bash
venv\Scripts\activate
```

### 3. Cài đặt thư viện cần thiết:
```bash
pip install openai
```

### 4. Chạy chương trình:
```bash
python main.py
```
